AI Image & Voice Generation Assignment

This project demonstrates the use of open-source generative AI tools for:

• AI Poster Generation using text prompts
• AI Voice Generation using gTTS
• AI Voice Assistant using Python and pyttsx3

Components:

- 5 AI-generated posters
- 3 generated voice outputs
- Simple voice assistant
- Research comparison report

Technologies Used:

- Python
- gTTS
- pyttsx3
- Stable Diffusion
